# Credit Card Financial Dashboard

## Project Overview

This project analyzes credit card transaction and customer data to provide real-time insights into revenue, transaction volume, customer demographics, and card performance. The dashboard was developed using SQL and Power BI to support data-driven business decisions.

## Project Objective

To develop a comprehensive credit card dashboard that provides insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

## Tools & Technologies

- SQL
- Power BI
- DAX
- Microsoft Excel
- Visual Studio Code

## Dataset Information

The project uses customer and transaction datasets containing:

- Customer demographics
- Credit card details
- Transaction information
- Revenue metrics
- Interest earned
- Customer spending patterns

## Project Workflow

1. Data Import and Cleaning
2. SQL Database Creation
3. Data Modeling
4. DAX Calculations
5. Dashboard Development
6. Business Insights Generation

## Dashboard Features

- Revenue Analysis
- Transaction Analysis
- Customer Demographics
- Card Category Performance
- Customer Segmentation
- Weekly Performance Monitoring

## Key Insights

- Overall Revenue: 57M
- Total Interest Earned: 8M
- Total Transaction Amount: 46M
- Male customers generated higher revenue than female customers.
- Blue and Silver cards contributed the majority of transactions.
- Texas, New York, and California contributed the highest revenue.
